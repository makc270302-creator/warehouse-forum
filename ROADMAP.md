# Roadmap

## MVP 1

- Подключить Supabase проект и выполнить SQL-схему.
- Создать первого администратора.
- Настроить Vercel и переменные окружения.
- Проверить вход, сводку, форум, документы и профиль.

## MVP 2

- Админ-панель управления пользователями. Готово.
- Импорт пользователей из Google Sheets через вставку таблицы. Готово.
- Добавить загрузку документов для РС и администраторов.
- Добавить комментарии к темам форума.
- Закрепление важных тем. Готово.

## MVP 3

- Добавить уведомления о важных объявлениях.
- Добавить поиск по форуму и документам.
- Добавить категории склада: приемка, хранение, отгрузка, инвентаризация.
- Добавить журнал действий администраторов.

## Production

- Подключить собственный домен.
- Включить ежедневные бэкапы Supabase.
- Настроить правила регистрации сотрудников.
- Проверить политики доступа перед запуском для всего склада.
